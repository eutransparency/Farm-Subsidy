This is the API on to the data the powers farmsubsidy.org. You might want to read about how we get the data, or download the raw data if you want to do more in depth analysis.

Output

All methods can be output in the following formats, by appending the 'format' GET parameter:

  • JSON (default) (for JSONP add 'callback' to the GET request)
  • XML
  • YAML

Throttling

Requests are throttled based on IP address, and unless otherwise stated are limited to 30 requests a minute.

If you need faster access to the data, you can contact us or download the data yourself
